时代不锈钢谈更换不锈钢- -最好的选择?
2012-5-17

当购买一个新的或更换为你的咖啡壶的卡,一般有两种可用的选项――玻璃或不锈钢(两者都有一点塑料盖子和处理)。玻璃盛通常比较便宜,但钢铁盛有一些益处的。


      玻璃盛是很容易破。几乎每个人都有有经验的撞了薄玻璃酒瓶的边路进攻的咖啡壶和将它粉碎。清洁的卡水槽中也可以是一个棘手的任务,因为它有时看来,即使是最轻微的挖掘与水槽的底部或另一盘会导致的卡打破。如果咖啡机是离开太久了,玻璃盛破解由于热,尤其是如果咖啡都蒸发了。


       玻璃盛不绝缘。这意味着有必要继续加热的咖啡一个加热元件来保暖。这往往会“over-cook“咖啡和使它日益激烈的更热。咖啡也变得更强的随着时间的过去,水蒸发。最后,更多电力用作加热元件必须继续保持咖啡温暖。


     不锈钢盛避免这个问题。因为显而易见的原因,不锈钢盛更难打破比玻璃盛。没有必要采取特别的措施在洗他们,他们却可以在多次(尽管这不是推荐!)没有伤害。不锈钢盛通常绝缘,类似于一个热水瓶。这使得咖啡保持原始热量从酿造和使其不必有加热元件不断。他们通常是密封的绝缘的一部分,所以水不会蒸发。


      尽管他们可以贵一些,比标准的玻璃咖啡用玻璃酒瓶,购买咖啡机用不锈钢酒瓶是一个不错的主意。即使你只买一个不锈钢置换酒瓶,你决定购买,这将帮你省钱在这两个替代和能源成本。另一个好处是,咖啡生产商利用不锈钢盛往往更时髦的比玻璃和塑料咖啡制造商。


     所有主要的电器公司生产不锈钢咖啡制造商和有种类繁多的定价和特性,从价格合理的百得DE790B的卡8-Cup热咖啡机,用不锈钢酒瓶在约35.00的价格不菲,但功能丰富BUNN Axiom DV TC -数码热的卡自动咖啡啤酒价格将超过600.00美元。（来自http://www.0768petmo.com/baidu.com）
